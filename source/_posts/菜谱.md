# 蒜蓉蒸排骨

蒜头剁碎点-蒜多点，炸点滚油泼上去，加入盐白糖生抽蚝油，搅匀淋在排骨上，一勺淀粉霍霍匀，腌制一小时，放入高压锅蒸55分钟

# 棍棍面

放豆瓣酱，蒜，洋葱炒香，下入配菜(豆芽，王中王火腿肠，橄榄，)如果放西红柿就多炒一会把它炒烂，然后加水，放入辣子，盐，花椒粉，放入面之后加点老抽

# 炒细面

炒香蒜放入豆芽，然后放入孜然，辣椒粉，生抽，水的量要控制好，然后倒入面霍霍匀

# 烤鱼

鱼切好(可以放平)，冰箱研制一晚(多点盐，料酒，生抽，葱，姜)，第二天拿出啦可以把表面水擦干，然后两面抹一层油，撒上盐，辣椒粉，孜然面

# 炒菜花

菜花焯水1-2分钟，油烧热，放入菜花，一定要最大火，放入盐炒出汁，烹入生抽，味极鲜

# 烧豆腐

豆腐切块，焯水(水烧开加入盐放入豆腐，焯久一点好吃)，锅中倒油放入蒜火锅底料炒出红油，加葱花炒香倒入适量水，放花椒粉，放入豆腐+老抽上色，把生粉加点水霍霍匀，分次倒入锅中收汁

# 火锅鸡

鸡腿切块(焯水)，葱姜下锅炒香，一勺郫县豆瓣，红油炒出来，一块火锅底料，放入干辣椒花椒麻椒大料香叶，小火编出香味，倒入鸡腿，然后加点料酒，加入生抽老抽，糖提鲜，加入水末过鸡腿，大火烧开，转小火盖上盖，15分钟，然后大火收汁出锅。

# 藤椒鸡抓

鸡爪剪去脚趾甲，刀劈成两半，下入锅中用白芷炖烂，或者可以用高压锅压烂(冷水下锅大火烧开撇去浮沫，捞出倒入高压锅炖个20分钟)，处理好的鸡爪泡冷水

花椒油(青花椒温泡水半小时)，青小米辣切成小段备用，菜籽油大火烧到冒烟降温到100°，放入前面的调料，小火炸到辣椒壳泛白(浸泡俩小时)

控干水份，倒入盆中放蒜泥，二斤条辣椒，红小米辣，白糖 ，鸡粉，盐，味精，白醋少量，生抽，辣椒油(好吃的辣椒油太关键了)，藤椒油，放点藤椒，霍霍匀

# 小炒肉

选用五花肉或者二刀肉，把皮用锅把皮烫一下然后用刀和水洗，冷水下锅,放入生姜葱段花椒料酒，中小火慢慢煮开，撇去浮沫用大火继续煮5-10分钟,煮到筷子可以插透猪皮而无血水流出即可，冲洗放凉后用到切成薄片,锅中倒油开小火把肉煸香突出油脂，下入姜辣椒炒香, 放点糖提鲜，到处然后煸一下青椒丝(这一步最好一开始热锅的时候搞)，倒入处理好的青椒丝，炒一下然后倒入木耳，烹入两圈生抽即可，可以来点红油

# 黄焖鸡

剁鸡块(不能太小)，放入清水中浸泡半小时,中途换水两到三次

泡过水的鸡块挤干水分，很关键,倒入葱姜料酒,加入刚才调好的酱料,腌制半小时

清洗干净的干香菇加入开水(这一步应该和第一步同时进行)，

八角3克，桂皮5克，香叶1克，小茴香2克，白芷3克，花椒3克，草果皮1个（去籽),放入料包，清水冲洗两到三遍

# 砂锅

**红油**: 首先给锅里倒入色拉油500克(我这里就用菜籽油),凉油下入大葱50克,洋葱50克,姜片20克,中小火慢慢炸,炸干炸黄捞出来(不能黑了),把火关掉等油温降到120°左右,下入豆瓣酱80克,辣妹子酱40克,黄椒酱25克,然后开小火炒出香油炒出红油,下入葱姜(碎)各15克大蒜(碎)克,下入冰糖10克(两块左右),炒到冰糖融合以后下入蒸鱼豉油200克,蚝油25克,香料粉15克(比例:八角8克,花椒4克,肉蔻5克,白芷2克,小茴香5克,桂皮3克,香砂5克,草果5克),小火烧开后慢慢熬上5分钟即可出锅

# 酸菜鱼，好吃就完事了，参考王刚和抖音
处理鱼骨和鱼片，首先在鱼骨中加入食用盐一勺(3g)，适量啤酒，搅拌一分钟加入清水把鱼骨清洗后捞出备用，处理鱼片，盆中加入适量的食用盐和啤酒，然后轻轻的将鱼片搅拌一分钟，搅拌至鱼片有黏手的液体时再加入清水清洗干净，必须把鱼片的上粘液清洗干净，捞出挤干水分备用，鱼片加入食用盐两克少许胡椒粉加入啤酒20克，然后用手朝着一个方向搅拌一分钟，必须搅拌至把所有调料和啤酒吸收为止，半个鸡蛋清搅拌均匀，加入土豆淀粉5克，搅拌均匀后研制10分钟备用，
酸菜200克切成小段备用，泡萝卜100克切成薄片备用，炮姜50克切成薄片备用，野山椒30克切碎备用，红泡椒切碎备用，
锅中放入适量清水，下酸菜焯水2分钟，2分钟后将酸菜到出备用(不要水)，锅洗干净把酸菜放入炒干水分(大火)，下入底油6成热把泡萝卜和炮姜野山椒下锅，再加入炒干水分的酸菜，中火所有辅料翻炒一分钟，到处放入碗中备用，锅清洗干净煎鱼骨头，加入适量的植物油滑锅，热油到出，再加入少许底油6成热把鱼骨头下锅，中火把鱼骨头煎两分钟，金黄色为止，所有辅料下锅开大火翻炒均匀，热水下锅，开大火烧开后撇去浮沫，再转中火煮3分钟，加入食用盐两克，白糖1克提鲜，再加入少许胡椒粉，最后加入少许白醋，把鱼骨和酸菜捞出放入盆中垫底，锅离火，把研制好的鱼片依次下锅，开小火划15秒左右，中途用炒勺轻轻推散，捞出放入盆中，鱼汤开大火烧开，汤底残渣分离，撒上切好的红泡椒和适量的葱花，泼明油少许，
# 小炒肉
后腿肉焯水，放八角香叶草果，切片放入生抽淀粉腌制15分钟左右，起锅烧油炒制变色放入料酒老抽大火炒1分钟左右放入白糖提鲜倒出，最好油少一点，放入线椒炒辣，倒入红油木耳，炒一会把肉倒入，放生抽再大火炒1分钟，出锅前到一点味极鲜或者味精

# 小炒肉2 
五花肉切片，生抽抓匀腌制15分钟，尖椒斜刀切长圈，热锅冷油翻炒到有一点点金黄，倒入白酒（料酒也行，一盖）翻炒，再倒入老抽（我用的一点酱油）翻炒一分钟倒出，用锅中剩下的底油下蒜皮爆香，倒入尖椒翻炒，抄到尖椒颜色变深，然后把五花肉倒入锅，一分钟倒两圈生抽调味（我没做这一步因为味够了），加一点白糖提鲜，再抄30秒（急活快抄）

# 捞汁
洋葱100克,香菜70克,姜90克,西芹160克,胡萝卜200克,鱼露120克,香辣红油230克,鸡精55克, 蚝油170克,冰糖180-200克, 鸡饭老抽70克,小米辣30-40克,青线椒80-100克,蒜片70克,海天生抽300克,味达美酱油200克,蒸鱼豉油300克,家乐鲜露100克,美极酱油100克,辣鲜露50克,纯净水烧开后下入香菜芹菜胡萝卜还有冰糖, 然后捞出菜,用菜水, 可以降低成本, 然后再加入其他的配料,切记不能加醋, 用的时候加一点芥末,然后倒入醋,再倒入捞汁
